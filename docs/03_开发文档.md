## 一、架构总览

### 1.1 设计哲学

AgentMemoryManager 的设计遵循以下原则：

1. **认知科学对齐：** 采用工作记忆 / 情节记忆 / 语义记忆 / 程序记忆的四层认知模型，与 Baddeley 工作记忆模型和 Generative Agents（2023）的实践对齐
2. **可插拔优先：** 存储后端、记忆策略、LLM 提供商均通过接口抽象，允许替换而不影响核心逻辑
3. **渐进式复杂度：** 默认配置下开箱即用；高级用户可逐层配置策略 Pipeline
4. **生产可观测：** 记忆系统的所有操作均可追踪，支持调试和审计

## 二、核心模块设计

### 2.1 记忆类型模型

#### 四层记忆体系

```python
# 认知科学映射：
# Working Memory  ←→  注意力焦点，容量有限，快速读写
# Episodic Memory ←→  个人经历事件，带时间戳，可检索
# Semantic Memory ←→  通用知识，实体关系，结构化
# Procedural Memory ←→  技能和操作模式，隐性知识
```

**WorkingMemory（工作记忆）**

```python
@dataclass
class WorkingMemory:
    session_id: str
    messages: list[Message]          # 原始对话轮次
    token_budget: int                # 最大 token 预算
    compression_ratio: float = 0.0   # 当前压缩比
    attention_sink_tokens: int = 4   # StreamingLLM 注意力汇 token 数
```

- **职责：** 管理当前活跃的对话上下文，确保不超出 token 预算
- **策略：** 优先使用 LLMLingua 压缩 + 注意力汇保留

**EpisodicMemory（情节记忆）**

```python
@dataclass
class MemoryRecord:
    id: str                          # UUID
    session_id: str
    user_id: str
    content: str                     # 原子事实或摘要
    source_messages: list[str]       # 来源消息 ID
    embedding: list[float]           # 向量表示
    importance_score: float          # 1-10，LLM 打分
    recency_score: float             # 衰减函数计算
    created_at: datetime
    accessed_at: datetime
    metadata: dict                   # 关键词、标签等
    links: list[str]                 # 关联记忆 ID（Zettelkasten）
```

- **职责：** 存储和检索跨轮次、跨会话的关键事件和事实
- **检索评分：** `score = α·recency + β·importance + γ·relevance`（参考 Generative Agents）

**SemanticMemory（语义记忆）**

```python
@dataclass
class Entity:
    id: str
    name: str
    entity_type: str                 # 人物 / 地点 / 概念 / 组织
    attributes: dict
    embedding: list[float]

@dataclass
class Relation:
    subject_id: str
    predicate: str
    object_id: str
    confidence: float
    valid_from: datetime
    valid_to: datetime | None        # None = 仍然有效（时序感知）
```

- **职责：** 维护知识图谱，支持实体关系的多跳查询
- **设计参考：** Zep/Graphiti 的时序感知图（Temporal Knowledge Graph）

**ProceduralMemory（程序记忆）**

```python
@dataclass
class ProcedureTemplate:
    id: str
    name: str
    description: str
    trigger_pattern: str             # 触发该流程的模式描述
    steps: list[str]                 # 步骤序列
    success_count: int
    embedding: list[float]
```

- **职责：** 存储成功的任务执行模板，支持 Agent 跨任务复用

#### 策略 Pipeline

```python
class StrategyPipeline:
    """将多个策略串联执行"""

    def __init__(self, strategies: list[MemoryStrategy]):
        self.strategies = strategies

    async def process(self, messages: list[Message], ...) -> ProcessResult:
        result = ProcessResult()
        for strategy in self.strategies:
            result = await strategy.process_input(messages, ...)
        return result

# 使用示例：
pipeline = StrategyPipeline([
    SummarizeStrategy(summarize_threshold=3000),
    AtomicFactsStrategy(),
    ReflectionStrategy(reflection_threshold=100),
])
```

#### Anthropic Claude 实现（默认推荐）

```python
import anthropic

class AnthropicClient(LLMClient):
    def __init__(
        self,
        model: str = "claude-sonnet-4-6",
        api_key: str | None = None,
    ):
        self.client = anthropic.AsyncAnthropic(api_key=api_key)
        self.model = model

    async def generate(self, prompt: str, **kwargs) -> str:
        message = await self.client.messages.create(
            model=self.model,
            max_tokens=kwargs.get("max_tokens", 512),
            system=kwargs.get("system", ""),
            messages=[{"role": "user", "content": prompt}],
        )
        return message.content[0].text
```

> 提示：对于 summarize / extract 等批量记忆操作，**强烈建议使用 Prompt Caching**（在 system prompt 前缀添加 `cache_control`），可显著降低延迟和成本。

## 四、关键算法设计

### 4.1 记忆检索评分

参考 Generative Agents（Park et al., 2023），综合三维度打分：

```python
def compute_retrieval_score(
    memory: MemoryRecord,
    query_embedding: list[float],
    current_time: datetime,
    weights: tuple[float, float, float] = (1.0, 1.0, 1.0),
) -> float:
    """
    综合评分：score = α·recency + β·importance + γ·relevance

    recency   : 指数衰减，半衰期约 1 天
    importance: LLM 打分，已归一化到 [0, 1]
    relevance : 与查询的余弦相似度
    """
    α, β, γ = weights

    # 近因性：指数衰减
    hours_elapsed = (current_time - memory.accessed_at).total_seconds() / 3600
    recency = pow(0.99, hours_elapsed)

    # 重要性：已存储在 MemoryRecord 中
    importance = memory.importance_score / 10.0

    # 相关性：余弦相似度
    relevance = cosine_similarity(memory.embedding, query_embedding)

    return α * recency + β * importance + γ * relevance
```

## 五、LangChain 集成适配器

```python
from langchain.memory import BaseMemory
from agent_memory_manager import MemoryManager

class AgentMemoryManagerAdapter(BaseMemory):
    """
    将 AgentMemoryManager 适配为 LangChain BaseMemory 接口。
    
    使用示例：
        memory = AgentMemoryManagerAdapter(
            manager=MemoryManager(...),
            session_id="user-123",
        )
        chain = ConversationChain(llm=llm, memory=memory)
    """

    manager: MemoryManager
    session_id: str
    token_budget: int = 2000
    memory_key: str = "history"

    @property
    def memory_variables(self) -> list[str]:
        return [self.memory_key]

    def load_memory_variables(self, inputs: dict) -> dict:
        query = inputs.get("input", "")
        context = asyncio.run(
            self.manager.build_context(
                query=query,
                session_id=self.session_id,
                token_budget=self.token_budget,
            )
        )
        return {self.memory_key: context}

    def save_context(self, inputs: dict, outputs: dict) -> None:
        messages = [
            Message(role="user", content=inputs.get("input", "")),
            Message(role="assistant", content=outputs.get("response", "")),
        ]
        asyncio.run(
            self.manager.add(messages=messages, session_id=self.session_id)
        )

    def clear(self) -> None:
        asyncio.run(self.manager.delete_session(self.session_id))
```

## 七、快速开始示例

```python
import asyncio
from agent_memory_manager import MemoryManager, MemoryConfig
from agent_memory_manager.backends import ChromaBackend
from agent_memory_manager.llm import AnthropicClient
from agent_memory_manager.embedders import OpenAIEmbedder
from agent_memory_manager.strategies import StrategyPipeline, SummarizeStrategy, AtomicFactsStrategy

async def main():
    # 1. 初始化组件
    config = MemoryConfig(
        llm_model="claude-sonnet-4-6",
        backend="chroma",
        episodic_strategy="atomic_facts",
    )

    manager = MemoryManager(
        backend=ChromaBackend(path="./chroma_db"),
        strategy=StrategyPipeline([
            SummarizeStrategy(summarize_threshold=3000),
            AtomicFactsStrategy(),
        ]),
        llm_client=AnthropicClient(model="claude-sonnet-4-6"),
        embedder=OpenAIEmbedder(model="text-embedding-3-small"),
        config=config,
    )

    session_id = "demo-session-001"

    # 2. 添加对话历史
    from agent_memory_manager.models import Message
    await manager.add(
        messages=[
            Message(role="user", content="我叫李明，是一名 Python 后端工程师，正在开发一个 AI 助手项目"),
            Message(role="assistant", content="你好李明！很高兴认识你。"),
        ],
        session_id=session_id,
    )

    # 3. 在后续对话中检索相关记忆并构建 prompt
    enhanced_prompt = await manager.build_prompt(
        base_prompt="请帮我推荐一些适合我当前项目的技术栈",
        session_id=session_id,
        token_budget=2000,
    )

    print(enhanced_prompt)
    # 输出将包含注入的记忆上下文：
    # ## 相关记忆
    # - 用户名为李明，是 Python 后端工程师
    # - 用户正在开发 AI 助手项目
    # ---
    # ## 当前请求
    # 请帮我推荐一些适合我当前项目的技术栈

asyncio.run(main())
```
